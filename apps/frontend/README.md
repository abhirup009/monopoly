# Frontend

Vite + React frontend for live Monopoly game visualization.

## Setup

```bash
cd apps/frontend
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and update if needed.

```
VITE_ORCHESTRATOR_HTTP_URL=http://localhost:3000
VITE_ORCHESTRATOR_WS_URL=http://localhost:3000
```
